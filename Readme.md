# AturUang
AturUang merupakan sebuah website edukasi manajemen keuangan dan smart budgeting yang dirancang untuk membantu generasi muda dalam mengelola keuangan secara mandiri, praktis, dan sesuai dengan gaya hidup digital masa kini.

## Fitur Utama
- Artikel edukasi tentang manajemen keuangan
- Sistem smart budgeting untuk pembagian dan perbandingan keuangan
- Visualisasi data dan rekomendasi sebagai hasil akhir

## Pengembangan
### Persyaratan
- Laptop
- Koneksi Internet
- Code Editor (VSCode, Sublime Text, dll.)
- Git
### Langkah Pengembangan
1. Clone repository ini
```bash
git clone https://github.com/Falrafa4/AturUang.git
cd AturUang
```
2. Install dependensi front-end
```bash
npm install
[npm run dev
npx serve public] secara bersamaan jika untuk development
```
3. Buka projek di code editor (contoh: Visual Studio Code)
```bash
code .
```
4. Buka terminal baru di code editor, jalankan untuk mulai mengembangkan projek dengan TailwindCSS
```bash
npm run dev
```
5. Jalankan server dengan npx atau dengan extension Live Server
```bash
npx serve public
```
6. Selamat ngoding :)
